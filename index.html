Swipe4Paws - Pet Adoption WebApp

The aim of the project “Swipe4Paws” is to create a full-stack web application where it is possible for people to see pets up for adoption in the country or in specific areas. This app makes it easier for potential adopters to see pets across multiple shelters in one place and thereby find a pet.

Tools used: Angular, Node.js, Express.js, postgreSQL, github

The "Healthy Desk" web app enhances workplace wellness by allowing users to control their desk’s height and personalize settings through individual profiles. Built in Angular, it promotes active work habits with gamified features, rewarding users with points and streaks for maintaining posture and regularly adjusting their desk.
